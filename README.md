# InvestHabits - 投资习惯训练

**中文版** | [English](./README_EN.md)

**InvestHabits (投资习惯)** 是一款专为开发者设计的 IDE 内嵌式投资心态模拟训练工具。通过 1-9 的随机数模拟市场波动，帮助你在繁忙的编码间隙，锻炼投资决策的果断性与风险控制意识。

---

## 🌟 应用介绍

InvestHabits 无缝集成在 IntelliJ IDEA 界面中，通过 Tool Window 提供清爽的模拟投资体验。它不仅是一个模拟器，更是一个心态磨炼场。通过不同的模拟环境，引导开发者进行冷静、智慧的投资决策。

### 核心亮点：
- **能力提升**：强调通过模拟训练，培养良好且稳胜的投资习惯，从而在潜移默化中提高个人的真实投资能力。
- **防沉迷与风控**：本工具的精妙设计在于通过设置**每日最高亏损限制**与**每日最高收益限制**，来防止用户头脑过热产生赌博心理。除了通过**结果分布网格**分析波动规律外，您还可以通过**记录**功能复盘每轮投资的详细操作。通过这些数据，您可以精准捕捉自己的操作细节，记录投资习惯的养成过程，并见证个人能力的不断进步。最终目的是让用户养成认真找出规律，保持冷静、智慧、稳定的操作习惯，靠长时间稳定的每日小收益不断积累成大财富。
- **结果分布**：直观的 6x6 网格展示今日所有操作的分布。绿色方格代表“涨”，红色代表“跌”，默认色代表“平”。右下角的“√”和“×”分别表示投资成功与失败。
- **逆向思维**：准确预判“平”局将获得 4 倍收益；但若选择“平”局失败，也将面临 4 倍的亏损风险。

## ✨ 功能特性

- **当前信息**：实时展示今日初始本金、当前余额、盈亏百分比及操作分布网格。
- **高额回报**：模拟市场中，准确预判“平”局将获得 4 倍于普通涨跌的收益回报，考验你的逆向思维。
- **投资周期**：支持从初始资金到破产或手动结算的完整投资轮次管理。
- **锁定机制**：一旦投资开始，关键设置将自动锁定，确保训练的严肃性与一致性。
- **帮助系统**：内置玩法指南，随时点击 `?` 图标查看详细规则。
- **多语言支持**：原生支持中文、英文、日文和韩文。

## 🛠️ 安装方式

1. 打开 IntelliJ IDEA。
2. 进入 `Settings` (Windows/Linux) 或 `Settings/Preferences` (macOS)。
3. 选择 `Plugins` 并在搜索框中输入 **InvestHabits**。
4. 点击 `Install` 并根据提示重启。
5. 安装完成后，在侧边工具栏找到 **InvestHabits** 图标即可开始训练。
